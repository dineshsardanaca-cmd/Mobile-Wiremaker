# **App Name**: MenuMorph

## Core Features:

- Dynamic Category Handling: Adapt the navigation system to display standard food categories and industry-specific beverage categories (On Tap, Seasonal Brews, Wine List, Happy Hour) based on the restaurant type.
- Variable Metadata Display: Toggle the display of metadata based on the selected item type (food or drink), showing relevant information such as Calories, Allergens for food, and ABV, IBU, Pour Size, Tasting Notes for drinks.
- Live Menu Scroll: Display a scrollable view with a sticky category header that allows quick navigation between food and drink sections.
- Beverage Card Design: Implement a specialized card layout for drinks that displays ABV%, volume, and origin/brewery information without appearing cluttered.
- Food Detail Screen: Showcase high-quality food imagery, allergen icons (Vegetarian, Vegan, Nut-Free, Gluten-Free), and dish descriptions on dedicated detail screens.
- Global Filter Overlay: Implement a slide-out filter menu allowing users to filter menu items by dietary restrictions or drink types.

## Style Guidelines:

- Primary color: Soft gray (#99A3A4) for neutrality and adaptability.
- Background color: Very light gray (#F0F4F5) for a clean backdrop.
- Accent color: Teal (#2E8B57) for subtle visual interest.
- Body font: 'PT Sans', sans-serif, suitable for the bulk of the text; Headline Font: 'Playfair', serif, for titles
- Use minimalist, scalable vector graphics (SVG) icons for allergens, dietary restrictions, and drink types to ensure clarity at various sizes.
- Employ a responsive grid system to adapt content to various screen sizes and orientations. Cards should have a clean, consistent layout with adequate spacing.
- Utilize subtle transition animations when navigating between categories or displaying the filter overlay.
