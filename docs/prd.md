### 20. Export Data to GitHub

*   **Feature Description:**
    Provide a way for "app owners" to sync the underlying data configurations (JSON files) to a personal GitHub repository. This allows for easy backups and version control of custom menus, product listings, and coupon data. The feature is located in the Account page and requires a GitHub Personal Access Token and repository path.

*   **Dependencies:**
    - GitHub API accessibility.
    - Server Actions for secure API requests.
    - Application data structured in JSON files (`menu-data.json`, etc.).

*   **Prompt:**
    ```
    I want to export my data to my github account. how can i do that?
    ```
